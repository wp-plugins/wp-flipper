It is very simple plugin. you can flip the wordpress content in any page or post. 

== Description ==

Just give the shortcode for flipping the content.

The following steps in given below for flipping the content.

Front side

Step1:Type the short code[flipper_front]

Step2:Type your content text or image.

Step3:Enclosing short code[/flipper_front]


Back side

Step1:Type the short code[flipper_back]
step2:Add your content text or images.
step3:Enclosing short code[/flipper_back]

There are three Parameters for setting the style in dynamic

parameter 1:width
parameter 2:height
parameter 3:bgcolor

For example

[flipper\_front width="200 height="200" bgcolor="red"] your content [/flipper\_front]


Note : In back content shortcode, you can set the paremeter for background color only.

parameter 1: bgcolor

For example

[flipper\_back bgcolor="red"] your content [/flipper\_back]

This plugin is used to flip the any wordpress content or image.

== Installation ==

== Installation ==

1. Upload \`wp-flipper\` to the \`/wp-content/plugins/\` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Done!
